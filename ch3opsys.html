Operating Systems Terminology

Term Defintion
operating system The software that controls the overall operation of the computer including file management, the user interface, and the program execution environment.
job queue a list of tasks to be carried out in turn by an operating system:
FIFO In computing and in systems theory, first in, first out (the first in is the first out), acronymized as FIFO, is a method for organizing the manipulation of a data structure (often, specifically a data buffer) where the oldest (first) entry, or "head" of the queue, is processed first.
interactive processing Interactive processing is a term that can have different meanings depending on the context. In general, it refers to the process of interacting directly with a program and/or data for immediately modifying, retrieving and/or displaying information.
time-sharing mutiprocessing Time-sharing, in data processing, method of operation in which multiple users with different programs interact nearly simultaneously with the CPU of a large-scale digital computer.
application software Application software is a computer program or a number of programs designed to help users perform specific tasks.
system software System software is software that provides a platform for other software.
utility software Utility software is a type of software that is designed to help users manage, maintain, and optimize their computer systems.
graphical user interface (GUI) The graphical user interface, or GUI , is a form of user interface that allows users to interact with electronic devices through graphical icons and audio indicators such as primary notation, instead of text-based UIs, typed command labels or text navigation.
shell In computing, a shell is a computer program which exposes an operating system 's services to a human user or other program.
kernel The kernel is a computer program at the core of a computer's operating system and generally has complete control over everything in the system.
device driver A device driver, also known as a driver or hardware driver, is a group of files that enable one or more hardware devices to communicate with the computer's operating system.
filer manager a program used to organize, list, and locate files and directories on a computer.
virtual memory memory that appears to exist as main storage although most of it is supported by data held in secondary storage, transfer between the two being made automatically as required.
boot strapping Bootstrapping is any test or metric that uses random sampling with replacement, and falls under the broader class of resampling methods.
ROM Read-only memory is a type of non-volatile memory used in computers and other electronic devices.
deadlock In concurrent computing, deadlock is any situation in which no member of some group of entities can proceed because each waits for another member, including itself, to take action, such as sending a message or, more commonly, releasing a lock.
spooling In computing, spooling is a specialized form of multi-programming for the purpose of copying data between different devices.
super user (or administrator) A superuser is an account with broad privileges, making it useful for system administration.
muti-factor authentication Multi-factor authentication (MFA) is a security mechanism that uses two or more distinct mechanisms to validate a user's identity
